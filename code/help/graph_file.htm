Формат .grph

Это особый формат, который хранит в себе полную информацию
о графе (точки, цвет и т.д.). С помощью этого формата удобно
хранить графы, в которых важна структура. Например, при
открытии такого файла, в котором хранится структура в виде 
дерева, откроется граф именно в виде дерева.

Данный формат открывается или сохраняется, создается и 
редактируется только в данной программе.
