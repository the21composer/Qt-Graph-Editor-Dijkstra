Поддерживаемые форматы

Программа может работать с различными входными и выходными данными:
.grph формат  Граф с сохранением форматирования.
.txt формат   Граф по списку инцидентности без форматирования.

Для открытия можно пользоваться стандартным "Файл->Открыть". 
Также можно использовать кнопки для открытия файла соответствующего формата.
Также, открыть граф можно с помощью перетаскивания - 
файл .txt или .grph необходимо перетащить в окно 
"Редактор графов".
