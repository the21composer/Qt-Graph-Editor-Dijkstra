Синтаксис ввода списка

Для создания пустой вершины необходимо ввести следующее:
	"имя вершины" :

Для добавления ребенка к вершине, после ":" необходимо 
через запятую перечислить вершины-дети:
	"имя вершины" : "ребенок 1", "ребенок 2"
	
Вершины можно дополнять детьми в любой момент. Например:
	"вершина" : "ребенок 1"
		........
	"вершина" : "ребенок n"
		
Не обязательно инициализировать все вершины в отдельных строках.
Например, граф
	"вершина" : "неиниц вершина"
создаст корректный граф.
