Максимальнй поток
Алгоритм Форда-Фалкерсона

s - исток, t - сток
f(i,j) - текущий поток через ребро (i,j)
c(i,j) - пропускная способность ребра (i,j)
cf(i,j) = с(i,j) - f(i,j)

0  Ford_Fulkerson(G, s, t)
1	for (для) каждого ребра (u, v) є E[G]
2	{
3           f(u,v)  = 0
4           f(v,u)  = 0
5	}
6	while существует путь p из s в t в остаточной сети Gf
7	{
8	     delt = min (cf(u, v) | (u, v) принадлежит р)
9
10           for (для) каждого ребра (u, v) из p
11           {
12		 f(u,v) = f(u,v) + delt
13                f(v,u) = f(v,u) - delt
14	     }
15	}



Для осуществления алгоритма поиска Максимального потока необходимо:

1)Выбрать исток - для этого на одной из вершин необходимо вызвать
контекстное меню и выбрать пункт "Установить как исток"

2)Выбрать сток - для этого на одной из вершин необходимо вызвать
контекстное меню и выбрать пункт "Установить как сток"

3)Выбрать во вкладке "Алгоритмы" Максимальный поток

4)Алгоритм можно остановить в любой момент или изменить его скорость
Для этого в левом верхнем углу появляется кнопка и слайдер

5)После завершения работы алгоритма на графе будет отображаться
анимация, изображающая поток. Для ее остановки в пункте "Алгоритмы"
выбрать "Очистить поток".
